
# VueJS Towers of Hanoi
[Repository](http://purchasepro.dthrllnrs.com/)


## Tech Stack

- **Framework:** VueJS
- **npm** npm v 21


## Run Locally

Install using npm

```bash
  npm install
```
```bash
  npm run dev
```
## Classes

| Class                                 | Description                               |
| ----------------------------------  | ----------------------------------------  |
| Game                 | The game class used to manage game state                        |
| Tower            | The tower class to represent the towers                       |
| Disc           | The disc class to represent the discs                             |
